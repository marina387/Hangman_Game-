\# Виселица (Hangman) - Игра на Python



Простая консольная реализация классической игры "Виселица" на Python.



\## Описание игры



Виселица (Hangman) - это классическая игра-угадайка, где игрок пытается отгадать загаданное слово, предлагая буквы. Всего 10 ходов. За каждую ошибку "строится" виселица.



\## Особенности



\- \*\*Текстовый интерфейс\*\* в консоли

\- \*\*Визуализация прогресса\*\* виселицы

\- \*\*Возможность вносить новые слова в файл .txt
